### <b>Install</b>

```ts
npm install @ricardo-jrm/date-format

yarn add @ricardo-jrm/date-format
```

<br />

### <b>Usage</b>

```ts
import { dateFormat } from '@ricardo-jrm/date-format'

dateFormat(new Date('2022-01-31'))
-> '12:00 AM - 31 Jan, 2022 GMT'
```
